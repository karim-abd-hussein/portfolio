# Karim Abd Hussein - Full-Stack Developer Portfolio

A professional portfolio website showcasing the skills, experience, and projects of Karim Abd Hussein, a Full-Stack Developer with expertise in EdTech and enterprise web applications.

## 🚀 Features

- **Modern Design**: Clean, professional layout with Bootstrap 5
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Smooth animations and transitions using jQuery and AOS
- **Sections**: Hero, About, Experience, Projects, Skills, Education, Contact
- **Contact Form**: Functional contact form with validation
- **Navigation**: Smooth scrolling navigation with active state highlighting
- **Timeline**: Visual timeline for work experience
- **Skill Bars**: Animated progress bars for technical skills
- **Project Cards**: Interactive project showcases with tech stacks

## 🛠 Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with Bootstrap 5
- **JavaScript**: Interactive features with jQuery 3.7
- **Bootstrap 5**: Responsive framework and components
- **Font Awesome**: Icon library
- **AOS**: Animate On Scroll library
- **jQuery**: DOM manipulation and event handling

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Local web server (optional, for development)

### Setup
1. Clone or download the project files
2. Place files in your web server directory (e.g., XAMPP htdocs)
3. Open `index.html` in your browser
4. Or access via local server: `http://localhost/portfolio`

### Development
- Modify `css/style.css` for styling changes
- Update `js/script.js` for JavaScript functionality
- Edit `index.html` for content changes
